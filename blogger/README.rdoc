== README

This exercise follows the open source Blogger 2 tutorial provided by JumpstartLab.

Rails basics learned:
1. Models, views, and controllers (MVC)
2. Data structures and relationships
3. Routing
4. Migrations
5. Views with forms, partials, and helpers
6. RESTful design
7. Adding gems for extra features
8. Authentication
